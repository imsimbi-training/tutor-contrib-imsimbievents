mongoeventtracker plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/imsimbi-training/tutor-contrib-mongoeventtracker 

Usage
-----

::

    tutor plugins enable mongoeventtracker
